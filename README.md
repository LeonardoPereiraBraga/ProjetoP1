## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo Android utilizando **Java**, com o objetivo de centralizar diversas funcionalidades em um único app — funcionando como um verdadeiro **Hub de Ferramentas**.

A aplicação integra módulos desenvolvidos anteriormente:

* 🎲 MegaSena
* ⚖️ Calculadora de IMC
* ✊✋✌️ Jokenpô
* 😊 Questionário da Felicidade (novo módulo)

O foco principal do projeto é praticar conceitos fundamentais do desenvolvimento Android, como:

* Navegação entre telas (**Activities**)
* Comunicação entre telas (**Intents**)
* Passagem de dados entre Activities
* Persistência de estado (ex: placar do Jokenpô)
* Lógica condicional baseada em pesos

---

## 🚀 Funcionalidades

### 🏠 Tela Inicial (Menu Principal)

* Interface central com botões para acesso aos módulos
* Navegação via **Intents explícitas**

---

### 🎲 MegaSena

* Geração de números aleatórios
* Simulação de jogo da Mega-Sena
* Exibição dos números sorteados

---

### ⚖️ Calculadora de IMC

* Entrada de peso e altura
* Cálculo automático do IMC
* Classificação (Abaixo do peso, Normal, Sobrepeso, etc.)

---

### ✊✋✌️ Jokenpô

* Escolha entre Pedra, Papel ou Tesoura
* Lógica de vitória/derrota
* Sistema de **placar persistente**
* Atualização em tempo real

---

### 😊 Questionário da Felicidade

* Perguntas com respostas ponderadas
* Sistema de cálculo baseado em **pesos**
* Resultado final indicando nível de felicidade
* Exibição de feedback ao usuário

---

## 🛠️ Tecnologias Utilizadas

* **Java**
* **Android Studio**
* **XML (Layouts)**
* **SDK Android**

---


## 🎯 Objetivo Educacional

Este projeto foi desenvolvido com o objetivo de:

* Consolidar conhecimentos em Android com Java
* Praticar arquitetura simples baseada em Activities
* Entender a comunicação entre telas
* Trabalhar com persistência local de dados
* Implementar lógica de negócio com base em regras
